General Functions
raise_admin_bar(): This function raises the admin bar if it is not raised. It takes in no parameters.

translate(): This function calls the Google Translate Cloud API in order to translate the page into the language specified by the user. It takes in two parameters, text and targetLanguage, which are used to specify the text to be translated and the language to translate the text into, respectively.

Home Page Functions
initVals(): This function initializes the values of variables used to render all the objects. It takes no parameters and simply uses the values set in the localStorage of the website to set the values of the variables.

initValsTiny(): This function has the exact same functionality as initVals(), but it is used for objects that are too large when rendered with initVals(), so it makes objects smaller.

print_All_Vals(): This function was used in debugging, but simply prints all values fetched from the backend in order to check if the values are correct.

getProduct(): Uses a fetch call in order to get the product information from the backend. It takes in no parameters and is based off the back-end getProduct() function.
Accessibility Functions
buttonClick(): Takes in a param param1 in order to determine which of the buttons was clicked on the screen, to determine what effect to apply to the screen. If param1 is 1, then the screen background is changed to a black and white background; if the parameter is 2, then the font size is increased. If the parameter is 3, then depending on which language is chosen, the screen is translated into that language after exiting the page.
